# Changelog

## v0.1.13

- 화면 응답 속도를 개선했습니다.
- DB/목록/문제점검/상태 정보를 짧게 캐시해 같은 화면을 열 때 반복 계산을 줄였습니다.
- 저장/수정/업데이트 후에는 캐시를 자동으로 비웁니다.

## v0.1.12

- 설정/관리 화면이 느려지는 문제를 줄이기 위해 모바일 주소/네트워크 조회 결과를 캐시 처리했습니다.
- 모바일 주소 조회 실패가 있어도 서버 화면이 느려지거나 멈추지 않도록 안정성을 개선했습니다.


## v0.1.11

- 모바일 접속 주소를 표시할 때 Windows 네트워크 정보 조회 출력 인코딩 때문에 서버 화면이 멈출 수 있던 문제를 수정했습니다.
- PowerShell 출력은 UTF-8/CP949를 모두 안전하게 처리하도록 변경했습니다.

## v0.1.10

- 문제 있는 항목 점검을 로그 탭에서 관리 탭으로 이동했습니다.
- 로그 탭은 최근 로그만 표시하도록 정리했습니다.
- 도움말의 문제 점검 안내 위치를 관리 탭 기준으로 수정했습니다.

## v0.1.9

- 모바일 접속 주소 표시를 개선했습니다.
- 실제 Wi-Fi/LAN 주소는 추천 주소로 표시하고, Hyper-V/WSL/Docker/VMware 같은 가상 어댑터 주소는 모바일 접속용이 아님을 따로 표시합니다.

## v0.1.8

- 모바일 접속이 Windows 방화벽에 막힐 때 사용할 수 있는 방화벽 허용/해제 스크립트를 추가했습니다.
- README와 서버 도움말의 모바일 접속 문제 해결 안내를 보강했습니다.

## v0.1.7

- 백그라운드 실행 시 실제로 열린 포트를 확인한 뒤 브라우저를 열도록 수정했습니다.
- 8787 포트가 막혀 8877 등으로 대체 실행될 때 잘못된 주소가 열리던 문제를 수정했습니다.
- 실행 중복 방지가 다른 프로그램의 8787 포트를 LocalReadLog로 오인하지 않도록 개선했습니다.

## v0.1.6

- 서버 종료 스크립트를 강화했습니다.
- PID, 포트, 작업관리자 프로세스 검색을 함께 사용해 백그라운드 서버를 더 확실히 종료합니다.

## v0.1.5

- 실행 중복 방지를 추가했습니다.
- 관리 탭에서 data 폴더, 백업 폴더, 로그 파일을 바로 열 수 있게 했습니다.
- 압축 해제 안내와 접속 포트 안내를 더 명확하게 정리했습니다.
- GitHub Release용 ZIP 이름을 정리했습니다.

## v0.1.4

- 로그 탭에서 줄바꿈이 보이지 않던 문제를 수정했습니다.

## v0.1.3

- 일부 환경에서 서버 화면이 제대로 표시되지 않던 문제를 수정했습니다.
- DB 저장 안정성을 개선했습니다.

## v0.1.2

- 설정, 관리, 로그 탭 구성을 정리했습니다.
- PC/모바일 접속 주소 링크와 복사 버튼을 개선했습니다.
- DB 가져오기와 복원 안내를 개선했습니다.

## v0.1.1

- 관리 화면 구성을 정리했습니다.
- DB 백업 목록 표시를 줄였습니다.

## v0.1.0

- 첫 공개 버전입니다.
- 브라우저 방문기록 스캔을 지원합니다.
- 작품별 최신 화수 관리를 지원합니다.
- 웹툰, 만화, 망가, 소설, 애니, 기타 분류를 지원합니다.
- 사이트와 브라우저 ON/OFF 설정을 지원합니다.
- 모바일 접속을 지원합니다.
- 접속 비밀번호 옵션을 지원합니다.
- 자동 업데이트를 지원합니다.
- DB 백업, 복원, 내보내기, 가져오기를 지원합니다.
- 백그라운드 실행과 Windows 시작 시 자동 실행을 지원합니다.
